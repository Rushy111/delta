About Umbrafiles

Version 2.0.0 — January 2026

What is Umbrafiles?

A private, peer-to-peer suite for secure file transfer, encrypted messaging, and remote device control. All connections are established directly between your devices using WebRTC.

🔐 Zero servers. Zero logs. Zero compromises.

Core Features

  • Secure Chat — Real-time encrypted messaging with file sharing
  • Remote Control — Bidirectional device control with low-latency input forwarding
  • Fast File Transfer — 128KB chunked P2P transfer with backpressure handling
  • Manual Signaling — Copy-paste or QR code pairing

Future Plans

While currently private, plans include automatic signaling, group connections, and public release with updated licensing.
