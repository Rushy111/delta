About Umbrafiles

Version 1.0.0 — Last Updated: January 2026

What is Umbrafiles?

Umbrafiles is a private, peer-to-peer (P2P) file transfer and streaming suite designed for seamless communication between your devices. It enables you to transfer files, mirror screens, and use your phone's camera as a webcam — all without any central server.

🔐 All connections are established directly between your devices using WebRTC technology. No data is ever stored on or routed through any third-party server.

Core Features

  • File Transfer — Send any file type between your PC and phone with real-time progress tracking and drag-and-drop support.
  • Screen Mirroring — Mirror your PC screen to your phone or receive a screen share from your phone with configurable quality settings.
  • Webcam Mode — Use your phone's camera as a high-quality webcam for your PC, with photo capture support.
  • End-to-End Encrypted — All data channels are encrypted using WebRTC's built-in DTLS/SRTP encryption.
  • No Server Required — Connections are established directly between peers using a manual signaling exchange.

Technology Stack

  • WebRTC — Real-time peer-to-peer data channels and media streams
  • Manual Signaling — Offer/Answer exchange via copy-paste or QR code
  • MediaStream API — Screen capture and camera access
  • File API — Client-side file handling and drag-and-drop
  • Pure HTML/CSS/JS — No frameworks, no dependencies, fully static

Why "Umbra"?

The word "umbra" refers to the fully shaded inner region of a shadow cast by an opaque object — symbolizing privacy, darkness, and the unseen. Umbrafiles operates in the shadows of your network, connecting your devices without leaving a trace on any server.

Future Plans

While Umbrafiles is currently a personal tool, there are plans to potentially distribute it in the future. A public version may include automatic signaling via a relay server, group connections, and additional streaming features. Until then, this remains a private, personal project.

Privacy Policy

Effective Date: January 1, 2026

Data Collection

Umbrafiles collects zero data. The application runs entirely in your browser with no external connections except the direct peer-to-peer link you establish. No analytics, no telemetry, no cookies, no tracking.

Data Storage

No data is stored on any server. All file transfers, screen mirroring, and webcam streams occur directly between your devices. When you close the browser tab, all connection data is immediately discarded.

Encryption

All peer-to-peer connections use WebRTC's built-in DTLS encryption for data channels and SRTP encryption for media streams. This is the same encryption standard used by leading secure communication platforms.

Network Information

The only network information processed is your device's IP address, which is inherently visible during a peer-to-peer WebRTC connection. This IP address is only visible to the peer device you directly connect to.

Third-Party Services

Umbrafiles does not use any third-party services, CDNs, analytics platforms, or advertising networks. The entire application is self-contained in a single HTML file.

For privacy-related inquiries, please contact the creator through private channels. As this is a personal project, there is no public support system.
