About

What is UmbraFiles?

Built for privacy-conscious individuals who demand the highest standards of security and simplicity.

🔐

Local-First Encryption

Every operation happens directly in your browser using the Web Crypto API. Your files never leave your device. No servers, no cloud, no middlemen. Your data stays yours — always.

🛡️

Military-Grade Security

We use AES-256-GCM encryption, the same standard trusted by governments and financial institutions worldwide. Combined with secure key derivation (PBKDF2), your data is protected against even the most sophisticated attacks.

🧹

Metadata Stripping

Images carry hidden data — GPS coordinates, device info, timestamps. Our metadata stripper removes all EXIF, IPTC, and XMP data, leaving you with a clean image that reveals nothing about its origin.

🔤

Text Scrambler

Send encrypted messages that only the intended recipient can read. Our text scrambler uses the same AES-256-GCM engine, providing end-to-end encryption for your conversations right in the browser.

⚡

Lightning Fast

Optimized for performance with WebAssembly-grade speeds. Large files are processed efficiently with streaming operations and real-time progress visualization so you always know what's happening.

🎨

Beautiful Experience

Privacy shouldn't mean sacrificing aesthetics. UmbraFiles features a stunning dark/light interface, smooth animations, custom cursor, and thoughtful micro-interactions that make security feel premium.

Privacy Policy

Last Updated: July 16, 2026

1. Information We Do NOT Collect

UmbraFiles is designed with privacy as its core principle. We explicitly do NOT:

  • Collect or store any files you upload or process
  • Store or transmit your encryption passwords or keys
  • Use cookies, tracking pixels, or analytics tools
  • Collect IP addresses, browser fingerprints, or device information
  • Create user accounts or require registration
  • Share any data with third parties (because we have no data to share)

2. How Your Data Is Processed

When you use UmbraFiles, all data processing occurs entirely within your web browser using the Web Crypto API, Canvas API, and standard JavaScript. Files you select for encryption, decryption, or metadata stripping are read from your device's storage and processed in memory. No data is ever transmitted over a network. When you download an encrypted file or clean image, it is generated locally and saved to your device.

3. Security Measures

We implement the following security measures:

  • AES-256-GCM Encryption: The gold standard for symmetric encryption, used by governments and financial institutions worldwide.
  • PBKDF2 Key Derivation: Passwords are converted to encryption keys using PBKDF2 with 100,000 iterations and random salts.
  • Secure Random Number Generation: All cryptographic nonces, IVs, and salts are generated using the Web Crypto API's secure random number generator.
  • Memory Safety: Processed data is not persisted to disk by the browser beyond what the user explicitly downloads.

4. Third-Party Services

UmbraFiles does not integrate with any third-party services, APIs, or CDNs. The entire application is self-contained. The only external resources are system fonts loaded by your operating system.

5. Children's Privacy

UmbraFiles is not specifically designed for children under 13. However, since no data is collected, there is no privacy risk associated with use by minors. Parental guidance is recommended for all users.

6. Your Rights

Since we do not collect any personal data, there is no personal data to access, modify, or delete. You have complete control over your data at all times.
